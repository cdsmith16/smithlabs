# Smith Labs

Professional portfolio and product showcase for Smith Labs.

## Tech Stack
- Next.js 14+
- React 18
- Deployed on Vercel
- Custom domain: https://smithlabs.llc

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment
Automatically deployed via Vercel on push to main branch.

## Products
- JoLo - Portfolio accountability tracker
- Present - Gift tracking system
- Link Strippa - URL management utility
- Card Benefits Map - Credit card benefits visualization
